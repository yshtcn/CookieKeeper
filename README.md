# 🍪 Cookie Keeper

一个 Chrome 浏览器插件，可定期在后台自动访问配置的网站，获取最新 Cookie 并保存到本地文件，供 yt-dlp 等工具使用。

## 功能

- **定时自动刷新**：按设定间隔（最短 1 分钟）在后台静默打开标签页，无需用户干预
- **多网站管理**：可配置任意数量的网站，每个独立设置刷新频率
- **自动写入文件**：运行后自动将 Cookie 覆写到下载目录的固定文件，无需手动操作
- **手动导出**：随时手动导出单个或全部网站的 Cookie
- **标准格式**：导出 Netscape HTTP Cookie File 格式，与 yt-dlp、curl、wget 完全兼容

## 安装

### 方式一：直接加载（推荐，无需账号）

1. 下载或克隆本仓库
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本仓库根目录（包含 `manifest.json` 的那一层）

## 使用

### 基本流程

1. 点击工具栏 🍪 图标，再点击 **⚙ 打开配置页**
2. 在 **全局设置** 中确认保存子目录（默认 `CookieKeeper`）
3. 点击 **＋ 添加网站**，填写网站地址和刷新间隔
4. 开启 **运行后自动写入固定文件**，文件名留空会自动生成
5. 保存后插件开始按间隔自动运行

> ⚠️ **重要**：Cookie 来自你当前的 Chrome 浏览器 Profile，请确保你已在 Chrome 中**登录**对应网站，否则只能获取到未登录的 Cookie。

### 自动保存路径

文件默认保存到：

```
系统下载目录 / CookieKeeper / 你设置的文件名.txt
```

例如（Windows）：

```
C:\Users\你的用户名\Downloads\CookieKeeper\youtube_com_cookies.txt
```

> Chrome 扩展受安全沙箱限制，只能写入**下载目录**内的路径，无法写入任意位置。

### 配合 yt-dlp 使用

```bash
# 下载需要登录的视频
yt-dlp --cookies "C:\Users\你\Downloads\CookieKeeper\youtube_com_cookies.txt" "https://www.youtube.com/watch?v=..."

# 或使用相对路径
yt-dlp --cookies cookies.txt "URL"
```

建议设置较短的刷新间隔（如 1～6 小时），使 Cookie 始终保持有效。

## Cookie 文件格式

导出文件为标准 **Netscape HTTP Cookie File**：

```
# Netscape HTTP Cookie File
# Generated by Cookie Keeper Chrome Extension
# Source:   https://www.youtube.com
# Exported: 2026-03-04T00:00:00.000Z

.youtube.com	TRUE	/	TRUE	1893456000	VISITOR_INFO1_LIVE	xxxxx
#HttpOnly_.youtube.com	TRUE	/	TRUE	0	__Secure-3PSID	xxxxx
```

## 注意事项

- Chrome 闹钟 API 最短间隔为 **1 分钟**
- 插件运行时会在后台短暂打开标签页，完成后自动关闭
- 部分网站有 CAPTCHA 或反爬机制，可能获取不到有效 Cookie
- Cookie 有效期因网站而异，建议根据实际情况调整刷新频率
